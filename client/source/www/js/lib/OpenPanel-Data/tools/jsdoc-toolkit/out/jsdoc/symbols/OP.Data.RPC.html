  • <static> OP.Data.RPC.bind(options)
    Requests a session id for given username and password. Calls success if all went well, calls error on http errors, invalid input, invalid output schema and all OpenPanel-Core errors. Throws Errors at missing username and/or missing password. 
    OP.Data.RPC.bind(
	{
		username : "openpanel-admin",
		password : "foobar",
		success : function (response) { },
		error : function (errorMessage, request, response) { }
	}
);
// username and password are required
//
    Parameters:
    {Object} options
    Information about the request
    {String} options.username
    The name of the user
    {String} options.password
    The password of the user
